Clone Repo
```bash
git clone https://github.com/elsuselamos/GetGrass.git
```
Open Directory 
```bash
cd GetGrass-Bot
```
Install Dependen
```bash
npm install
```
Run Multi Script With Proxy
```bash
npm start
 ```
or
```bash
node index.js
```
Run single With Proxy or No proxy
```bash
node index-normal.js
```
Normal mode is Working for No Proxy
